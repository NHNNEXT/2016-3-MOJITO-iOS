#### git flow / 트렐로 연동 계획
- "card 1(트렐로)" = "feature 1(git flow)" = "기능 1(기획)"
- "check list 1(트렐로)" = "세부 기능 1(기획)"

- "check list 1(트렐로)"을 완성하면, "commit(git flow)" 할 것_푸쉬는 자유롭게(어자피 기능은 홀로 개발)

- "기능 1(기획)" (or "card 1(트렐로)") 을 완성하면 "finish feature(git flow)"할 것
